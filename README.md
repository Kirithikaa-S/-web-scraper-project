# Book Information Web Scraper

## Project Description

This project is a Python-based web scraper that extracts book information from the Books to Scrape website. The scraper collects book titles, prices, and ratings and stores the data in a CSV file.

## Features

* Connects to a website using Requests
* Parses HTML using BeautifulSoup
* Extracts book titles
* Extracts book prices
* Extracts book ratings
* Saves data into a CSV file

## Technologies Used

* Python
* Requests
* BeautifulSoup4
* CSV Module

## Project Structure

web_scraper_project/
├── main.py
├── books.csv
├── requirements.txt
└── README.md

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the program:
   python main.py

## Output

The scraper creates a books.csv file containing:

* Title
* Price
* Rating
